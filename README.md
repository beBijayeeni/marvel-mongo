## Imp Pipenv commands
- **Install pipenv package**:

    ```bash
    pip install pipenv
    ```


- **Initialize a virtual environment**:

    ```bash
    pipenv install
    ```

- **Start the pipenv terminal**:

    ```bash
    pipenv shell
    ```
- **Install required in your venv**:

    ```bash
    pipenv install -r requirements.txt
    ```
